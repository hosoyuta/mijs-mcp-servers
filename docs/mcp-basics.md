# MCP (Model Context Protocol) 基礎

## MCPとは

Model Context Protocol (MCP) は、AIモデルとツールやデータソースを接続するためのプロトコルです。

## MCPサーバーの役割

MCPサーバーは以下の機能を提供します：

- **ツール**: AIが呼び出せる機能を提供
- **リソース**: AIが参照できるデータを提供
- **プロンプト**: 定型的なプロンプトテンプレートを提供
